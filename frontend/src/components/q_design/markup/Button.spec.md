# Figma Properties

### Hierarchy=Primary, Secondary, or Tertiary
Hierarchy=Primary: these buttons with a solid fill should be the principal call to action on the page, and help the user focus on the main flow
Hierarchy=Secondary: these buttons have less emphasis than primary, and are used for CTAs not as critical as the principal call to action
Hierarchy=Tertiary: these buttons have even less emphasis than secondary buttons, and are used for smaller tasks that do not demand much attention
Type=Text or Icon Only
Type=Text: these buttons display written text that state what the button does
Type=Icon Only: these buttons do not display any text and only show an icon
Icon=True or False
Icon=True: an icon displays alongside the button text
Icon=False: only button text displays, with no icon
Action=Productive or Destructive
Action=Productive: do not delete, cancel, or remove any information and will be the majority of the buttons we use
Action=Destructive: trigger destructive actions including Delete, Remove, or Stop, and is styled in red as a warning (aside from the disabled state)
State=Default, Hover, Focus, Active, or Disabled
State=Default: when the button is not being hovered or clicked
State=Hover: when user hovers over the button (desktop only)
State=Focus: when the user is using something that is not a mouse to navigate through the page (usually a keyboard) and we show which element they are currently focused on as they tab through the page
State=Active: the moment the user clicks on the button
State=Disabled: when a button intentionally cannot be interacted with
